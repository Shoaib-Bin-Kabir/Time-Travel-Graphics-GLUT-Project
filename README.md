# Time Travel Graphics (GLUT) Project

## Project Overview

This project is an interactive graphics simulation built with GLUT/OpenGL. It presents multiple themed eras and a portal-based transition system for scene switching and keyboard interaction.

## Main Focus

- Interactive era-based simulation
- Keyboard-controlled animations and transitions
- Scene design using C++ and GLUT

## Repository Structure

- Time Travel Graphics (GLUT) Project
- Apocalyptic_Era: Source files and build/project files for apocalyptic scene
- Egyptian_Era: Source files and build/project files for egyptian scene
- Medieval_Era: Source files and build/project files for medieval scene
- Time_Portal: Portal scene and switching logic
- Screenshots: Captured images from each scene
- Project_Details.txt: Controls and usage instructions

## Notes

Project_Details.txt contains scene controls and instructions to run and navigate between scenarios.
